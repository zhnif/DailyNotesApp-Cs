# Daily Notes App 📝

A simple Windows Forms App (.NET Framework) project for students.  
This app helps you write and save daily notes with a local SQL Server database.

---

## ✨ Features
- Add new notes with title and content
- Save notes into local SQL Server database (`NotesDB.mdf`)
- View list of notes in a DataGridView
- Double-click a note to read its full content
- Simple and beginner-friendly design

---

## 🛠️ Tech Stack
- C# (.NET Framework, Windows Forms)
- SQL Server LocalDB
- ADO.NET for database operations

---

## 📂 Project Structure
- `Program.cs` → Entry point
- `Form1.cs` → Main form logic
- `Form1.Designer.cs` → UI layout
- `NotesDB.mdf` → Local database

---
